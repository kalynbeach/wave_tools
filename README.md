# wave-tools

> Audio and music production tools ~

## Usage

---

(Example) Get the BPM of a 16-bit mono WAV file:

`cargo run -- bpm <file path>`

Example:
`cargo run -- bpm samples/test_beat_m.wav`

List Ableton Live Projects:

`cargo run -- projects`

List Ableton Live Templates:

`cargo run -- templates`
