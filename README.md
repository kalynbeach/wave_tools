# wave_tools

> Audio and music production tools ~

## Usage

---

Get the BPM of a 16-bit mono WAV file:

```bash
cargo run -- bpm <file path>
```

- Example: `cargo run -- bpm samples/test_beat_m.wav`

List Ableton Live Projects:

```bash
cargo run -- projects
```

List Ableton Live Templates:

```bash
cargo run -- templates
```
